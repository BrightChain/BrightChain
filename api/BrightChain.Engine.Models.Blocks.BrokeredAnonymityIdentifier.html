Class BrokeredAnonymityIdentifier

Constitues really just the user's id- however the system will allow users to store data with registered identities or even anonymously but we will store error correction (FEC) data that allows us to recover the original Id if all the pieces of it (once sharded) are reconstructed. Plan is to use Reed Solomon.
