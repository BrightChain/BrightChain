Class BrightChain

Brightened data chain, can be composed of file-based CBLs or brightened ChainLinq based data blocks. Although a BrightChain contains brightened data, the CBL block itself is not brightened. TODO: improve memory usage. Don't keep full copy, do all on async enumeration?
