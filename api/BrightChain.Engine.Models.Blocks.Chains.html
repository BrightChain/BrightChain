Represents a virtual map of all the contituent tuple-sets/blocks in a given source/reconstructed file. TupleStripe

A tuple stripe is a representation of the blocks used only to Recover a source block. In each row, only the last block is expected to be the non-randomizer block, but order doesn't matter within a stripe.
